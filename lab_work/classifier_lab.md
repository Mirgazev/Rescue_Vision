# Лабораторная работа: SceneAnalyzer

Цель работы — изучить извлечение интерпретируемых признаков изображения и реализовать простой классификатор условий видимости.

Признаки:

1. Средняя яркость.
2. Стандартное отклонение яркости.
3. Дисперсия Лапласиана.
4. Плотность границ.
5. Диагональный коэффициент градиента.

Результат работы — файл `features.csv` и простое дерево решений для режимов NIGHT, FOG, SMOKE, RAIN и OFF.
